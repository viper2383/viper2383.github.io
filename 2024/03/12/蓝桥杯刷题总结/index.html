字典序

Java中Arrays.sort()的三种常用用法（自定义排序规则）_arrays.sort自定义排序-CSDN博客

Java Lambda 表达式 | 菜鸟教程 (runoob.com)

这题主要学到了一点：

这段代码使用 Java 中的 Lambda 表达式作为参数来对数组 arr 进行排序。看一下这段代码的作用：

  1. Arrays.sort(arr, (a, b) -> {...});：这是一个使用 Lambda 表达式的排序方法调用。它对数组 arr 进行排序，其中 Lambda 表达式 (a, b) -> {...} 用于定义排序规则。
  2. (a, b) -> {...}：这是一个 Lambda 表达式，表示一个函数式接口。在这里，(a, b) 是 Lambda 表达式的参数列表，用于比较数组中的两个元素。{...} 内的代码是 Lambda 表达式的主体，用于定义比较规则。
  3. String ab = a + b; 和 String ba = b + a;：这两行代码将数组中的元素 a 和 b 转换为字符串，并分别将它们拼接起来，得到两个新的字符串 ab 和 ba。
  4. return ab.compareTo(ba);：这一行比较了字符串 ab 和 ba 的字典顺序。如果 ab 在字典中应该排在 ba 之前，则返回负数；如果它们相等，则返回零；如果 ab 在字典中应该排在 ba 之后，则返回正数。这样的比较方式可以实现按照自定义规则对数组进行排序。

综上，这段代码的作用是对数组 arr 中的元素进行自定义规则的排序，规则是将两个元素转换为字符串后，按照字典顺序比较它们的组合。
