蓝桥杯2022年第十三届省赛真题-求和

题目描述

给定 n 个整数 a1, a2, · · · , an ，求它们两两相乘再相加的和，即 S = a1 · a2 + a1 · a3 + · · · + a1 · an + a2 · a3 + · · · + an-2 · an-1 + an-2 · an + an-1 · an.

输入格式

输入的第一行包含一个整数 n 。

第二行包含 n 个整数 a1, a2, · · · an。

输出格式

输出一个整数 S，表示所求的和。请使用合适的数据类型进行运算。

样例输入

样例输出

提示

对于 30% 的数据，1 ≤ n ≤ 1000，1 ≤ ai ≤ 100。

对于所有评测用例，1 ≤ n ≤ 200000，1 ≤ ai ≤ 1000。

代码

思路：

首先读取数组A[]，计算数组的前缀和，用数组的前缀和和数组运算得到结果

前缀和就是a[i]前面元素的和。记presum[]为前缀和数组，

presum[1]=a[0]，presum[2]=a[1]+a[2]=presun[i-1]+a[2]，……，presun[i]=presum[i-1]+a[i];

  • 设数组的和为S。
  • \(a1*a2+a1*a3+....+a1*an=a1*(S-a1)=a1(S-presum[1]);\)
  • \(a2*a3+a2*a4+...+a2*an=a2*(S-a1-a2)=a2*(S-presum[2])....\)
  • \(到an时，那一串和等于：an*(S-presum[n]);\)
  • 但是因为数组是从0开始的，所以上面的a1，a2，a3……an下标依次为0，1，2，……n-1

题目 2682:

蓝桥杯2022年第十三届省赛真题-GCD

题目描述

给定两个不同的正整数 a, b，求一个正整数 k 使得 gcd(a + k, b + k) 尽可能大，其中 gcd(a, b) 表示 a 和 b 的最大公约数，如果存在多个 k，请输出所有满足条件的 k 中最小的那个。

输入格式

输入一行包含两个正整数 a, b，用一个空格分隔。

输出格式

输出一行包含一个正整数 k。

样例输入

复制

样例输出

复制

提示

对于 20% 的评测用例，a < b ≤ 105 ； 对于 40% 的评测用例，a < b ≤ 109 ； 对于所有评测用例，1 ≤ a < b ≤ 1018 。

代码

  1. 根据性质：若a>=b，有gcd ( a , b ) = gcd ( a - b , b ) = gcd ( a , a - b )
  2. 同理，将a+k 、b+k 代入gcd ( a , b ) 可得 gcd ( a+k , b+k ) = gcd ( a - b , b+k ) = gcd ( a+k , a - b )
  3. 此时目标：“使gcd ( a+k , b+k )最大”转化为“使 gcd ( a - b , b+k )或gcd ( a+k , a - b )最大”，要使后者gcd最大，很显然这个gcd的最大值肯定是a-b （a>b)。因此题目意思变为：求使得 (a+k) % (a-b) ==0 的最小k值
